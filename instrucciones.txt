Para evitar problemas con las migraciones al momento de hacer el merge: 
------------------------------------------------
-Borrar la base de datos actual o crear una nueva, para posteriormente poblarla mediante el migrate.
	-Si vas a poblar una base de datos nueva hay que cambiarla en el settings.py 

-Borrar todo el contenido de la carpeta "migrations" excepto el archivo "init.py"
-Volver a crear la migracion mediante la consola (makemigrations y migrate) -->volverá a crear la migracion "initial.py"
-Ya realizado lo anterior ir a la consola nuevamente y crear nuevamente el superusuario con los nuevos campos. (username, nombres, apellidos,
										                 rut, dv, correo, telefono y contraseña)
-Iniciar sesion en el administrador de django para verificar que se creo correctamente.
